Sobre o Projeto AutoFeeder

O AutoFeeder é um sistema desenvolvido para automatizar a alimentação de pets, trazendo mais praticidade e bem-estar tanto para os tutores quanto para os animais.

Materiais Utilizados

  • Arduino Uno
  • Microservo motor
  • Sensor de peso
  • Sensor de nível de ração
  • Bluetooth HC-05
  • Protoboard
  • Fios jumper
  • Fonte de alimentação

Etapas do Projeto

Etapa Descrição
Planejamento Definição das funções do AutoFeeder e levantamento de materiais.
Desenvolvimento Montagem do circuito no Arduino e programação das funções.
Testes Validação dos sensores e ajustes na quantidade de ração liberada.
Documentação Criação deste site e do relatório explicativo do projeto.

Ferramentas Digitais Utilizadas

  • TinkerCAD — Simulação de circuitos
  • Fritzing — Design do circuito
  • Editor de código: VSCode
  • Drivers e IDE Arduino — Programação da placa
